# DONPONLINE Website

A responsive one-page website for Don P's umbrella brand.

## Included sections
- Hero / featured release
- Shows and podcast appearances
- New music and catalog
- Beat store
- Music business advice
- Production tutorials
- About Don P
- Newsletter and social links

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/favicon.svg`

## How to preview
Open `index.html` in a browser.

For the cleanest local preview, run a simple web server from the folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## What to replace
Search `index.html` for `data-placeholder`. Replace each `href="#"` with the actual link:
- YouTube channels and playlists
- Spotify / Apple Music / YouTube Music
- BeatStars, Airbit, Shopify, or another beat store
- Instagram, TikTok, X, and Facebook

Also replace:
- `booking@donponline.com`
- `business@donponline.com`

Connect the newsletter form to Mailchimp, Kit, Beehiiv, Substack, or another platform.

## Suggested domain and setup
- Domain: `donponline.com`
- Hosting: Netlify, Vercel, Squarespace custom-code area, Shopify, or traditional web hosting
- Analytics: Google Analytics or Plausible
- Email: `booking@donponline.com`, `business@donponline.com`
